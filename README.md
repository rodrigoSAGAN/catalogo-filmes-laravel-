O projeto Catálogo de Filmes Interativo é uma aplicação web desenvolvida em Laravel com o objetivo principal de demonstrar a aplicação de boas práticas de Engenharia de Software e Qualidade. O foco do desenvolvimento não se limitou à criação de um sistema CRUD (Criar, Ler, Atualizar e Deletar) básico para filmes, mas sim em garantir a integridade, a confiabilidade e a manutenibilidade do código.

Os objetivos de qualidade foram claros: assegurar a integridade do Model, garantindo que regras de negócio como o limite de rating de 1 a 5 sejam estritamente aplicadas; validar o fluxo completo do CRUD, assegurando que todas as interações do usuário com o banco de dados funcionem de forma coesa e segura; e validar a usabilidade (UX/UI), proporcionando uma interface intuitiva com tema Dark Mode e design responsivo utilizando Tailwind CSS.

Para alcançar essa confiabilidade, foi adotada uma estratégia de testes completa, dividida em três níveis. Os Testes Unitários, realizados com PHPUnit, focam na lógica de negócio do Model Movie, validando campos obrigatórios, a regra de range do user_rating, o comportamento de campos opcionais e a correta aplicação de Query Scopes para filtros inteligentes e o fallback automático de URLs de imagem.

Os Testes de Integração/Feature, também realizados com PHPUnit, simulam o fluxo da aplicação do ponto de vista do usuário autenticado, validando as requisições HTTP e a correta manipulação dos dados no banco para as operações de Listagem, Criação, Atualização e Exclusão. Por fim, foram realizados Testes Manuais, documentados separadamente, que incluíram um Teste de Sistema (End-to-End) para validar o fluxo completo da aplicação (Login $\rightarrow$ Criar $\rightarrow$ Listar) e um Teste de Usuário (Avaliação Heurística) para avaliar a usabilidade, o contraste do tema Dark Mode e a intuitividade do recurso Drag and Drop. O projeto foi aprovado em todas as etapas de teste.
